How Application Works -> Step-By-Step

& 1

Program welcomes User into the program and gives a first question, if he/she wants to see rules, how to use application in the most better way.

& 2

YES NO
If user types 'YES' then program will display the application rules If user types 'NO' then program will continue to another step

& 3

Program asks User next Question and User has to choose - in which source he/she wants to count the quantity of word occurrences. User has 2 choices: input or file, but also he/she can type special keyword - 'stop' and close the running program permanently!

& 4

Based on User answer, following actions will be executed

ANSWER EXPLANATION RESULT
INPUT
  • user enters: input keyword in the console.
  • user enters some text in the console.
  • user enters desired word, which quantity will be counted.
  • final answer will be printed out by the program.
FILE
  • user enters: file keyword in the console.
  • user enters the exact/correct location of target file.
  • user enters desired word, which quantity will be counted.
  • final answer will be printed out by the program.
FILE
  • user enters: file keyword in the console.
  • user enters the wrong/invalid location of target file.
  • program gives feedback and asks user to enter valid location again.
  • user can type special keyword stop and close the program from here as well.
STOP
  • user enters: stop keyword in the console.
  • The execution of program will be stopped, program will print out final farewell message.
OTHER
  • user enters: any other word - beside => input, file, stop.
  • Program asks user again and again until, he/she enters one from following keywords: input, file, stop
Main functionality of - TextHandler class
Method How It works Code
replace_punctuations method takes one string parameter - text and replaces every punctuation symbols with empty string
count_word_quantity method takes two string parameters: text and word, then uses 'replace_punctuations' method to clean provided text and finally returns the number of occurrences of provided word in provided text.
Main functionality of - LargeFileHandler class
Method How It works Code
check_file_existance method checks, if file exists on location provided by user.
split_file method takes file_path as a string parameter and splits found large .txt file into smaller chunks - 10K size. Files will be located in split_files directory.
clean_split_file_directory method cleans split_files directory from newly created small files/chunks. After executing this method - that files will be removed permanenetly.
get_file_name_list method checks split_files directory and returns the names of all .txt file as a list.
replace_punctuation method takes one string parameter - text and replaces every punctuation symbol with empty string.
count_word_quantity method takes two string parameters: file_path and word. This method executes 'split_file' and 'get_file_name_list' methods, then iterates through each small .txt file and counts the number of occurrences of provided word in each file. Finally method will sum-up counting results, executes 'clean_split_file_directory' method and returns final quantity.
